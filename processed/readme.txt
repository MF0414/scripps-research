processed files from the raw data

lstDir.txt: a directed adjcency list of who follows who.  id's refer to our own id's that are assigned 0 to n-1

lst.txt is an undirected adjacency list of the users.  A link is created between i and j if i follows j or j follows i

tweets.txt is a file of tweets gathered from two days in June of 2017.

attr.txt is the attribute file.  Each line shows the original twitter id for the user, the user name and the number of followers.
